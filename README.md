# Application-Tracker
A website which helps you track your job, internship, or apprenticeship applications.

You can access the website over [here](https://grumyv2.github.io/Application-Tracker/).

## Features
- Add applications with company, role, status, deadline, location, and notes.
- Filter your applications by status.
- Dashboard summary showing total, applied, interview, and offer counts.
- Data persists in the browser via localStorage.
- Delete individual entries or reset all.
